virtual bool ProtocolInterface::SendEncapsulatedData ( const std::string &  targetmask,
const std::string &  cmd,
const CommandBase::Params &  params,
User *  source = NULL 
)
inlinevirtual

Send an ENCAP message to all servers matching a wildcard string. See the protocol documentation for the purpose of ENCAP.

Parameters
targetmaskThe target server mask (can contain wildcards)
cmdThe ENCAP subcommand
paramsList of string parameters which are dependent on the subcommand
sourceThe source of the message (prefix), must be a local user or NULL which means use local server
Returns
Always true if the target mask contains wildcards; otherwise true if the server name was found, and the message was sent, false if it was not found. ENCAP (should) be used instead of creating new protocol messages for easier third party application support.
