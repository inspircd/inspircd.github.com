◆ Displayable()

const std::string & KLine::Displayable ( )
overridevirtual

Returns the displayable form of the pattern for this xline, e.g. '*@foo' or 'baz'. This must always return the full pattern in a form which can be used to construct an entire derived xline, even if it is stored differently internally (e.g. GLine stores the ident and host parts separately but will still return ident@host for its Displayable() method).
